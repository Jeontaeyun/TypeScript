# 서버리스란?

## 서버리스 \| Server Less란?

 **서버의 특정 기능\(이미지 압축, API\)을 클라우드 서비스가 대신 관리해줘서 개발자나 운영자가 서버를 관리하는 데 드는 부담을 덜어주는 것**을 말합니다. 

 서버리스 컴퓨팅을 할 때는 AWS EC2와는 다르게 VM 인스턴스를 미리 구매해야 합니다. 단순한 코드를 업로드 한 뒤, 사용량에 따라 요금을 지불하면 됩니다.
