# 인터페이스

## 인터페이스 Interface

 타입스크립트에서만 지원하는 특징으로 인터페이스는 **타입이며 컴파일 후에는 사라집니다**.  

추상 클래스는 선언과 구현이 모두 존재하지만 인터페이스는 선언만 존재하며, 멤버 변수와 멤버 메소드를 선언할 수 있지만 접근 제한자는 설정할 수 없습니다. 

자식 인터페이스는 여러 부모 인터페이스를 다중 상속할 수 있습니다. 

```typescript
interface 인터페이스이름 {
    멤버변수이름 : 타입;
    name: string;
    age: number;
    job : string;
}
```
